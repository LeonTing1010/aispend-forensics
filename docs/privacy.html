Privacy Policy

AI-spend Forensics - Last updated: April 29, 2026

Summary: This extension does not collect, transmit, or share any user data with external servers. All data is stored locally on your device.

Data Collection and Usage

AI-spend Forensics is designed with privacy as a core principle:

  • ✅ No data collection - We don't collect any personal information
  • ✅ No data transmission - Nothing is sent to external servers
  • ✅ No data sharing - Your data never leaves your browser
  • ✅ No tracking - No analytics or tracking scripts

Local Data Storage

All data is stored locally on your device using Chrome's storage API. This includes:

  • Extracted spend data from AI tool dashboards
  • Extension settings and preferences

Permissions

Permission Purpose
storage Save spend data locally between browser sessions
notifications Alert users of spending anomalies (optional feature)
activeTab Access AI tool dashboard pages for data extraction
host permissions Read spend data from cursor.com, claude.ai, replit.com

Remote Code

This extension does not use any remote code. All code is bundled within the extension package and reviewed by the Chrome Web Store team.

Data Access

The extension only reads data from pages you actively visit:

  • Cursor settings/billing pages (cursor.com)
  • Claude settings/billing pages (claude.ai)
  • Replit billing pages (replit.com)

Single Purpose

This extension has a single purpose: to help users track and monitor their AI tool spending across multiple platforms by reading publicly visible billing information from AI tool dashboards.

Compliance

This extension complies with the Chrome Web Store Developer Program Policies regarding data usage and privacy.
