# AI-spend Forensics

**Track your AI tool spending across Cursor, Claude, and Replit.**

Know where your money goes. One popup, all platforms, zero setup.

## Install

**[Install from Chrome Web Store](https://chromewebstore.google.com/detail/ai-spend-forensics/EXTENSION_ID)** — free, one click.

Or install manually from [latest release](https://github.com/LeonTing1010/aispend-forensics/releases): unzip → `chrome://extensions` → Developer mode → Load unpacked.

## Supported Platforms

- **Cursor** — cursor.com/settings
- **Claude** — claude.ai/settings/billing  
- **Replit** — replit.com billing

## Features

- Automatically detects when you visit AI tool dashboards
- Displays current spending in a clean popup
- Shows spending across multiple platforms at a glance
- Badge icon shows total spend
- All data stays on your device (local storage only)

## How It Works

1. Install the extension
2. Visit your AI tool dashboard pages
3. Extension automatically reads and stores spend data
4. Click the extension icon anytime to see all your spending

## Privacy

- All data stays on your device (local storage only)
- No data is sent to external servers
- No tracking or analytics
- Only reads data from pages you visit
